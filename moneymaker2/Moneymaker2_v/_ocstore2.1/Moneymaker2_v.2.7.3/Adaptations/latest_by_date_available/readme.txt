Изменения в логику работы вывода Последних товаров под шаблон Moneymaker 2 (Opencart 2.1/2.3)
Opencart 2.1: mmr2.latest_by_date_available21.ocmod.xml
Opencart 2.3: mmr2.latest_by_date_available23.ocmod.xml

Установка:
1. Вы должны установить единственный ocmod-файл модуля для внесения изменения в системные файлы движка, для этого в админке необходимо перейти в раздел Дополнения - Установка дополнений, и загрузить ocmod.xml-файл модуля
2. После этого перейдите в менеджер дополнений (Дополнения - Менеджер дополнений) и убедитесь что добавленный файл модуля появился там (должен быть один новый файл - Moneymaker 2 Latest By Date Available Opencart 2.1/2.3), если все ок, нажмите справа вверху кнопку Обновить, чтобы выполнить обновление кеша модификаций.
3. Модуль установен

Использование:
Модуль предназначен для возможности настраиваемого вывода товаров в модуле Последние, по-умолчанию в движке этот модуль выводит последние добавленные товары, с помощью данной модификации модуль будет выводить те товары, у которых указана более "свежая" дата поступления в настройках товара (Date Available), таким образом можно прямо в админке товаров изменять дату поступления нужным товарам и контролировать работу модуля (что невозможно при использовании исходного модуля, поскольку дата добавления товаров недоступна к редактированию из админки)
