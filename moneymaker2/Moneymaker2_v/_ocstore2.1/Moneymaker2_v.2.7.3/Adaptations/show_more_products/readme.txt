Адаптация скрипта для подгрузки товаров без перезагрузки всей страницы под шаблон Moneymaker 2 (Opencart 2.1/2.3)
Ссылка на скрипт: https://opencartforum.com/files/file/2459-%D0%BF%D0%BE%D0%BA%D0%B0%D0%B7%D0%B0%D1%82%D1%8C-%D0%B5%D1%89%D0%B5-showmore/

Установка:
1. Скрипт по ссылке ставить не нужно, вы должны установить единственный ocmod-файл модификации для внесения изменения в системные файлы движка, для этого в админке необходимо перейти в раздел Дополнения - Установка дополнений, и загрузить ocmod.xml-файл mmr2.showmoreproducts.ocmod.xml
2. После этого перейдите в менеджер дополнений (Дополнения - Менеджер дополнений) и убедитесь что добавленный файл модуля появился там (должен быть один новый файл - Moneymaker2 Show More Products, если все ок, нажмите справа вверху кнопку Обновить, чтобы выполнить обновление кеша модификаций.
3. Модификация установена

Использование:
Модификация добавляет ко всем блокам пагинации на сайте дополнительный блок с кнопкой Показать еще, по нажатию которой на текущую страницу будут подгружаться товары со следующей страницы без необходимости переходить на следующую страницу.
Если вы используете модули, меняющие вывод товаров на собственный (например, фильтры товаров), то данная модификация может не работать, т.к. ей не будет ничего известно о выводе товаров фильтром.
